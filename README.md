# K8s in-place resize examples

Example apps in various languages to demonstrate the impact of k8s pod in-place resize on the application runtime. This repository is for **demo purposes only**, not for production code.

## Prerequisites

* a [kind](https://kind.sigs.k8s.io/) cluster (modify `build.sh` scripts and the various resize scripts if you use another k8s cluster / container registry)
* bash
* kubectl
* helm
